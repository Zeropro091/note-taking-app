# Freelancer Portfolio Builder

**Category:** Career Tools
**Effort:** Low
**Monetization:** One-time payment

## Description
A simple, elegant portfolio builder specifically for freelancers with integrated project case studies, testimonials, and inquiry forms.

## Key Features
- Drag-and-drop portfolio builder
- Project case study templates
- Client testimonial management
- Inquiry form with email notifications
- Custom domain support
- SEO optimization
- Analytics dashboard
- PDF resume generation

## Tech Stack
- Frontend: Next.js 14 + Tailwind
- CMS: Sanity or Supabase
- Auth: NextAuth.js
- Email: Resend
- Hosting: Vercel

## Revenue Model
- Free: subdomain.portfol.io
- Pro: $49 one-time - Custom domain, remove branding
- Plus: $99 one-time - All features + premium templates
