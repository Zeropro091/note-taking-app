# Skill Sharing Platform

**Category:** Education
**Effort:** High
**Monetization:** Commission

## Description
Connect people who want to teach skills with those who want to learn. One-on-one video lessons or small group classes.

## Key Features
- Teacher profiles and reviews
- Skill categories and search
- Video lesson scheduling (Zoom integration)
- Payment processing
- Lesson recordings
- Calendar management
- Messaging between teachers/students
- Commission-based revenue

## Tech Stack
- Frontend: Next.js + React Native
- Backend: Node.js/Python
- Video: Zoom API / Daily.co
- Payments: Stripe Connect
- Database: PostgreSQL
- Real-time: WebRTC for chat

## Revenue Model
- Commission: 15-20% per lesson
- Teacher: Sets own prices ($20-100/hour typical)
- Platform fee on top of lesson price
- Featured listings: $10/month for teachers
