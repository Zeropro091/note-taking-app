# Micro-SaaS Directory

**Category:** Developer Tools
**Effort:** Low
**Monetization:** Affiliate + Sponsored

## Description
Curated directory of micro-SaaS products with pricing, revenue estimates, and founder stories.

## Key Features
- Product listings with details
- Founder interviews and stories
- Revenue estimates and metrics
- Voting and reviews
- Categories and filters
- Newsletter with new additions
- Submit your own SaaS
- Acquisition marketplace

## Tech Stack
- Frontend: Next.js
- Backend: Supabase
- Database: PostgreSQL
- Payments: Stripe for featured listings
- Newsletter: Substack or custom

## Revenue Model
- Featured listings: $99/month
- Sponsored newsletter spots: $250/issue
- Affiliate: Commission on SaaS tools
- Job board: $50/job posting
