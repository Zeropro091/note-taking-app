# Personal Recipe Manager

**Category:** Food
**Effort:** Medium
**Monetization:** Freemium

## Description
Digital recipe box with AI-powered meal suggestions, nutritional analysis, and smart shopping lists.

## Key Features
- Recipe import from URLs (auto-parsing)
- Photo upload for each recipe
- Nutritional information calculator
- Meal planning calendar
- Smart shopping list from meal plan
- Recipe scaling for servings
- Search by ingredients you have
- Share recipes with family
- Print cookbooks

## Tech Stack
- Frontend: Next.js + React Native
- Backend: Node.js/Supabase
- Recipe Parsing: Custom scraper or RecipeAPI
- Nutrition: Edamam API / USDA database
- Storage: AWS S3 for images

## Revenue Model
- Free: 50 recipes, basic features
- Pro: $4.99/month - Unlimited recipes, nutrition, meal planning
- Family: $7.99/month - 5 users, sharing
