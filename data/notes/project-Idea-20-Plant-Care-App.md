# Plant Care Reminder

**Category:** Lifestyle
**Effort:** Low
**Monetization:** Freemium

## Description
Never forget to water your plants again. Track your plant collection with customized care schedules.

## Key Features
- Plant library with care instructions
- Custom care schedules (water, fertilizer, mist)
- Photo gallery for each plant
- Care history log
- Push notifications for care tasks
- Plant identification from photo
- Weather-based suggestions
- Community Q&A
- Shop for plants and supplies

## Tech Stack
- Frontend: React Native
- Backend: Node.js/Supabase
- Plant ID: PlantNet API or custom model
- Database: PostgreSQL
- Notifications: OneSignal

## Revenue Model
- Free: 10 plants
- Pro: $2.99/month - Unlimited plants, ID features
- Marketplace: Affiliate commission on plant sales
