# Simple Budget App

**Category:** Finance
**Effort:** Medium
**Monetization:** Freemium

## Description
Easy budget tracking with automatic categorization, spending insights, and bill reminders.

## Key Features
- Bank sync via Plaid
- Automatic transaction categorization (AI)
- Budget categories with alerts
- Bill due date reminders
- Spending insights and trends
- Savings goals with progress
- Multiple budgets (personal, business)
- Export for taxes
- Net worth tracking

## Tech Stack
- Frontend: React Native
- Backend: Node.js/Python
- Bank Sync: Plaid API
- AI: Custom categorization model
- Database: PostgreSQL
- Notifications: OneSignal

## Revenue Model
- Free: Manual entry, 1 bank account
- Pro: $4.99/month - Bank sync, AI categorization
- Plus: $9.99/month - Multi-account, investing tracking
