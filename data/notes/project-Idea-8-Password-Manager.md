# Password Manager

**Category:** Security
**Effort:** High
**Monetization:** Subscription

## Description
Secure password manager with biometric auth, breach monitoring, and secure sharing for families/teams.

## Key Features
- Military-grade encryption (AES-256)
- Biometric unlock (Face ID, fingerprint)
- Password generator with customizable rules
- Dark web breach monitoring
- Secure password sharing
- Digital legacy (emergency access)
- 2FA code storage (TOTP)
- Cross-platform sync

## Tech Stack
- Frontend: React Native + Electron
- Backend: Node.js
- Database: End-to-end encrypted (user never sends master key)
- Encryption: crypto / Web Crypto API
- Auth: SRP protocol (zero-knowledge)

## Revenue Model
- Free: 1 device, 50 passwords
- Premium: $24/year - Unlimited devices, breach monitoring
- Family: $48/year - 5 users, sharing
- Business: $5/user/month
