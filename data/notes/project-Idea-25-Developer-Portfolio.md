# Developer Portfolio Generator

**Category:** Career Tools
**Effort:** Medium
**Monetization:** One-time payment

## Description
AI-powered developer portfolio generator that pulls from GitHub, LinkedIn, and writes project descriptions.

## Key Features
- GitHub sync (projects, contributions)
- AI-generated project descriptions
- LinkedIn integration for work history
- Skill extraction from code
- Multiple beautiful themes
- Custom domain support
- Blog section
- Analytics dashboard
- Resume export (PDF)

## Tech Stack
- Frontend: Next.js
- Backend: Node.js/Python
- GitHub: GitHub GraphQL API
- AI: OpenAI for descriptions
- Database: PostgreSQL (Supabase)
- Hosting: Vercel (for user portfolios)

## Revenue Model
- Free: subdomain.portfol.io
- Pro: $29 one-time - Custom domain
- Plus: $59 one-time - AI descriptions, blog, analytics
- Enterprise: Custom branding
