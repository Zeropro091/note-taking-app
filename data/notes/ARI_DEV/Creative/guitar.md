# 🎸 Guitar Progressions & Ideas

---

## 🎵 Favorite Progressions

### 1. The Emotional Journey
```
Am - F - C - G
```
**Mood**: Nostalgic, hopeful
**Use**: Ballads, emotional songs

### 2. The Upbeat Lift
```
C - G - Am - F
```
**Mood**: Uplifting, pop
**Use**: Happy songs, anthems

### 3. The Dark & Moody
```
Em - C - G - D
```
**Mood**: Intense, dramatic
**Use**: Rock, alternative

### 4. The Jazz Walk
```
ii7 - V7 - Imaj7
```
**Mood**: Smooth, sophisticated
**Use**: Jazz, R&B
