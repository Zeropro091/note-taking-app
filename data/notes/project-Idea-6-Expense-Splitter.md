# Group Expense Splitter

**Category:** Finance
**Effort:** Low
**Monetization:** Freemium

## Description
Split bills with roommates, friends, or travel groups. Track who owes whom and settle up easily.

## Key Features
- Bill splitting by amount, percentage, or shares
- Receipt scanning (OCR)
- Multiple currencies with real-time conversion
- Expense categories and photos
- Payment reminders
- Venmo/PayPal/Stripe integration for settlements
- Group chat for each expense
- Export to Excel/CSV

## Tech Stack
- Frontend: React Native
- Backend: Node.js/Supabase
- OCR: Google Vision API / Tesseract.js
- Payments: Stripe Connect / PayPal API
- Database: PostgreSQL (Supabase)

## Revenue Model
- Free: 5 active groups
- Pro: $2.99/month - Unlimited groups, receipt scanning
- Ads: Optional for free users
