# Closing Techniques - The Art of Getting to Yes

## The Golden Rule of Closing

**If you've done the discovery well, closing should feel natural for both parties.**

A close is simply **the next logical step** in a conversation where value has been established.

---

## Pre-Closing Checklist

Before attempting any close, confirm:
- [ ] You understand their primary pain point
- [ ] They've acknowledged they have a problem
- [ ] They've expressed interest in solving it
- [ ] You've addressed their main objections
- [ ] You know who makes the decision
- [ ] You know their timeline

**If any box is unchecked → More discovery needed, not better closing.**

---

## Essential Closing Techniques

### 1. The Trial Close
Test readiness without asking for commitment.

```
"How does this sound so far?"
"Does this approach make sense to you?"
"Is this the kind of solution you had in mind?"
```

**If they say yes** → Proceed to actual close
**If they say no/unsure** → Ask: "What would make this better for you?"

### 2. The Assumptive Close
Proceed as if the decision is already made.

```
"When would you like us to start?"
"Which address should we use for the contract?"
"Should we schedule the kickoff for this week or next?"
```

**Best when**: Strong buying signals throughout conversation

### 3. The Option Close
Offer choices that both lead to commitment.

```
"Would you prefer the monthly or annual plan?"
"Should we include the training add-on or stick with core?"
"Would you like to handle billing now or at the end of the month?"
```

**Key**: Both options are things you want, not "buy or not buy"

### 4. The Urgency Close
Create gentle pressure based on timeline.

```
"We have 3 spots left for this month."
"The pricing increases on the 1st."
"If we start this week, we can include [bonus]."
```

**Must be genuine** or it backfires.

### 5. The Summary Close
Recap everything, then ask for commitment.

```
"So to summarize what we've covered:
- You're struggling with [problem]
- You need [solution] by [timeline]
- Our approach does [specific benefits]

Does that sound like the right path forward?"
```

### 6. The Question Close
Get them to sell themselves.

```
"What would need to be true for you to move forward?"
"If we could solve [specific objection], would you be ready?"
"On a scale of 1-10, how ready are you to proceed?
 (If not 10) What would make it a 10?"
```

### 7. The Silence Close
State your close, then **stop talking**.

```
"I believe this is the right solution for you.
Based on everything we've discussed,
I'd recommend we move forward with the Pro plan.
[Silence... wait for them to speak]"
```

**Why it works**: Most people fill silence. If you stay quiet, they talk themselves into it.

### 8. The "No Pressure" Close
Remove pressure to build trust.

```
"I don't want you to decide today. Think it over.
Sleep on it. If it still makes sense in a week,
let's move forward. If not, no worries at all."
```

**Why it works**: Paradoxically increases conversion by showing confidence.

## Closing for Service Businesses (Indonesia Context)

See [[Local-Business-Optimization]] for pricing insights.

### For Web Development Services
```
"I recommend we start with Phase 1 (landing page)
for IDR 2M, deliver in 2 weeks.
Once you see the results, we can discuss Phase 2.

Should I prepare the invoice for this week or next?"
```

### For Ongoing Services
```
"We can do the monthly plan at 500K/month,
or save 20% with quarterly payment.

Which works better for your cash flow?"
```

### For Custom Projects
```
"I'll put together a detailed proposal
with exact timeline and deliverables.
Can we review it together on Thursday?
10am or 2pm work better?"
```

## The "No" and What To Do

### "I need to think about it"
```
"Totally understand. What specifically would you
like to think about? Is it the fit, the timing,
or something else I can help clarify?"
```

### "It's too expensive"
```
"I hear that. Can you help me understand what
you're comparing it to? What's the budget range
you're working with?"
```

### "Let me check with my partner/boss"
```
"Of course. What information would be helpful
for that conversation? I can put together
a summary document for you."
```

### "Call me next month"
```
"Would it be okay if I follow up in 3 weeks
with any updates or new information?
In the meantime, I'll send you some resources."
```

## Never Close Until...

- [ ] You know their real budget
- [ ] You know their timeline
- [ ] You've spoken to the decision maker
- [ ] You understand their decision process
- [ ] You've identified all stakeholders

**The close happens at the beginning. The rest is just filling in the form.**
